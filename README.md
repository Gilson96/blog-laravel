## Blog-Article
An article blog built with Laravel, where users can apply CRUD operations on an article and comment.

You can view the site live here: https://blog-article-9e7b5e9272c9.herokuapp.com/

## Technologies
- HTML, 
- Bootstrap,
- Php,
- Laravel,
- Blade
- API
- Postman,
- Git/ GitHub,
- Heroku
